This Python program is designed to scrape data from the website "https://scrapeme.live/shop". 
It uses the requests and lxml modules to extract data from the website and saves it to a JSON file.

This program requires Python 3 and the requests and lxml modules. You can install these modules using pip by running the following commands:
# pip install requests
# pip install lxml



To use this program, simply run the Scrapeme_live.py file from the command line
The program will then scrape the website and save the extracted data to a JSON file named 'Pokemon.json'. 
The data includes the product name, price,stocks,SKU,description,categories and tags

If you encounter any issues while using this program, please feel free to open an issue on the GitHub repository.
